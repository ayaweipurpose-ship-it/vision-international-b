## Before You Start

Make sure you have Node.js installed:
- Download from: https://nodejs.org
- Get the LTS version (recommended)

## 3 Simple Steps

### Step 1: Extract the ZIP File
- Right-click the ZIP file
- Choose "Extract All"
- Remember where you extracted it!

### Step 2: Open Terminal in the Folder

**Windows:**
1. Open the `vision-final` folder
2. Click in the address bar
3. Type `cmd` and press Enter
4. A black window opens - this is your terminal!

**Mac:**
1. Open the `vision-final` folder
2. Right-click and choose "New Terminal at Folder"

### Step 3: Run These Commands

Copy and paste these commands ONE AT A TIME:

```bash
npm install
```
*Wait for this to finish (might take 1-2 minutes)*

Then:
```bash
npm run dev
```

### Step 4: Open Your Browser

You'll see a message like:
```
Local: http://localhost:5173/
```

Click that link OR open your browser and go to:
```
http://localhost:5173
```

## üéâ Success!

Your website should now be running!

## What Each Command Does

- `npm install` - Downloads all the packages needed (like React)
- `npm run dev` - Starts your website in development mode

## To Stop the Server

Press `Ctrl + C` in the terminal

## To Start Again

Just run:
```bash
npm run dev
```
