## ğŸ¨ Design Principles Used

1. **Consistent Spacing** - Everything uses multiples of 4px or 5px
2. **Visual Hierarchy** - Important things are bigger and bolder
3. **Color Psychology** - Green = growth, trust, health
4. **White Space** - Plenty of breathing room
5. **Hover Feedback** - Users know what's clickable
6. **Shadows** - Create depth and dimension
7. **Rounded Corners** - Modern and friendly

## ğŸ’¡ CSS Tips for Beginners

### Gradients
```css
background: linear-gradient(135deg, #colorStart 0%, #colorEnd 100%);
```

### Shadows
```css
box-shadow: 0 4px 15px rgba(0,0,0,0.1);
/* horizontal vertical blur color */
```

### Transitions
```css
transition: all 0.3s ease;
/* what duration timing */
```

### Hover Effects
```css
.card:hover {
    transform: translateY(-5px);  /* Move up 5px */
    box-shadow: 0 8px 30px rgba(0,0,0,0.15);
}
```
